# Nazaka - Business Overview

## What is Nazaka?

Nazaka is a B2C e-commerce mobile application built with Flutter, targeting the Libyan market with a focus on Arabic-first user experience.

## Product Vision

Provide a seamless, localized shopping experience for Libyan consumers with:
- Arabic-first interface (with English support)
- Local payment methods
- Libyan market-specific features
- Fast, native mobile experience

## Target Users

- **Primary**: Libyan consumers shopping on mobile
- **Age**: 18-45
- **Platform**: iOS and Android
- **Language**: Primarily Arabic, secondary English

## Key Differentiators

1. **Arabic-First**: RTL design, Arabic typography, culturally adapted UI
2. **Local Market Focus**: LYD currency, local shipping, Libyan addresses
3. **Mobile-Optimized**: Flutter native performance
4. **Modern Design**: Material 3, smooth animations, premium feel

## Business Model

- B2C e-commerce marketplace
- Commission-based revenue (TBD)
- Multiple vendors (future)
- Direct shipping integration

## Success Metrics

- User engagement (DAU/MAU)
- Order completion rate
- Average order value (AOV)
- Cart abandonment rate
- Customer satisfaction (NPS)
