# Nazaka - Spacing

> Nazaka's spacing tokens, border radii, and elevation levels.

---

## Spacing Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `xs` | 4dp | Tight spacing between related elements |
| `sm` | 8dp | Small gaps, list item padding |
| `md` | 16dp | Standard padding (most common) |
| `lg` | 24dp | Section spacing, card padding |
| `xl` | 32dp | Large gaps between sections |
| `xxl` | 48dp | Page margins, large spacing |

---

## Border Radius (Material 3 Shapes)

| Shape | Radius | Usage |
|-------|--------|-------|
| **Extra Small** | 4dp | Chips, small elements |
| **Small** | 8dp | Buttons, text fields |
| **Medium** | 12dp | Cards (default) |
| **Large** | 16dp | Bottom sheets, dialogs |
| **Extra Large** | 28dp | FABs |

---

## Elevation Levels

Material 3 uses tonal elevation (color shift) instead of heavy shadows:

| Level | Elevation | Usage |
|-------|-----------|-------|
| **0** | 0dp | Flat surfaces, backgrounds |
| **1** | 1dp | Cards, sheets |
| **2** | 3dp | Menus, dropdowns |
| **3** | 6dp | FABs, snackbars |
| **4** | 8dp | Navigation drawers |
| **5** | 12dp | Modal bottom sheets |
