# Nazaka - Color Palette

> Nazaka's brand color values for Material 3 ColorScheme.

---

## Brand Colors

**Primary**: Purple (#924DBF) - Main brand identity
**Secondary Container**: Light purple (#F5E7FF) - Navigation indicator background
**Outline**: Gray (#D1D5DC) - Borders and dividers

---

## Light Theme

| Role | Hex | Description |
|------|-----|-------------|
| **Primary** | `#924DBF` | Main brand color, primary buttons, FABs |
| **OnPrimary** | `#FFFFFF` | Text/icons on primary color |
| **PrimaryContainer** | `#F3DAFF` | Tonal surfaces, chips |
| **OnPrimaryContainer** | `#320054` | Text on primary container |
| **SecondaryContainer** | `#F5E7FF` | Navigation bar indicator background |
| **Surface** | `#FFFFFF` | Card backgrounds, sheets |
| **OnSurface** | `#1C1B1E` | Primary text, icons |
| **SurfaceVariant** | `#E8E0EB` | Dividers, disabled states |
| **OnSurfaceVariant** | `#4A454E` | Secondary text |
| **Secondary** | `#655A6F` | Secondary actions |
| **Tertiary** | `#80525A` | Accents, highlights |
| **Error** | `#BA1A1A` | Error states, destructive actions |
| **Outline** | `#D1D5DC` | Borders (cards, buttons, text fields) |
| **Background** | `#FFFBFF` | App background |

---
