# Nazaka - Typography

> Nazaka's font choices and text scale.

---

## Fonts

| Locale | Font Family | Usage |
|--------|-------------|-------|
| English (`en`) | **Roboto** | Default Material 3 font |
| Arabic (`ar`) | **Almarai** | Arabic typography with good readability |

Fonts are loaded via `google_fonts` package.

---

## Typography Scale

Material 3 text styles with sizes and weights:

| Style | Size | Weight | Usage |
|-------|------|--------|-------|
| `displayLarge` | 57sp | 400 | Hero text (rare) |
| `displayMedium` | 45sp | 400 | Large headings |
| `displaySmall` | 36sp | 400 | Section headings |
| `headlineLarge` | 32sp | 400 | Page titles |
| `headlineMedium` | 28sp | 400 | Section titles |
| `headlineSmall` | 24sp | 400 | Card titles |
| `titleLarge` | 22sp | 500 | App bar titles |
| `titleMedium` | 16sp | 500 | List item titles |
| `titleSmall` | 14sp | 500 | Subtitles |
| `bodyLarge` | 16sp | 400 | Primary body text |
| `bodyMedium` | 14sp | 400 | Default body text |
| `bodySmall` | 12sp | 400 | Captions |
| `labelLarge` | 14sp | 500 | Button text |
| `labelMedium` | 12sp | 500 | Chips, badges |
| `labelSmall` | 11sp | 500 | Small labels |
